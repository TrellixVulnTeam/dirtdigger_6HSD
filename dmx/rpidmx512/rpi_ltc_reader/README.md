## Raspberry Pi SMPTE TimeCode Reader / LTC - Protocol Converter ##

Supported input :

-     SMPTE LTC
-     MIDI
-     ArtNet 

Supported output :

-     MIDI
-     ArtNet
-     Displays :      
	-     Console (external monitor)
	-     OLED display 128 x 64 SSD1306 I2C
	-     LCD 16x2 I2C (auto-detect)
	-     8x 7-segment display (MAX7219)
	-     MAX7219 Matrix display
